Studio Update v2.8.1 Patch Notes

TS TechSupport Posted 1 week ago

This maintenance release addresses several issues reported by our developer community and includes performance improvements across the board.

Bug Fixes

Script Editor

  • Fixed an issue where scripts would sometimes fail to save properly when using auto-save
  • Resolved a rare crash that could occur when undoing changes in large Lua files
  • Fixed syntax highlighting for certain Lua patterns

Property Grid

  • Improved performance when working with instances that have many properties
  • Fixed alignment issues with numeric input fields
  • Resolved an issue where some properties wouldn't update visually after being modified via script

Performance Improvements

We've optimized several core systems:

  • Asset loading: Reduced memory usage when importing multiple assets
  • Rendering: Improved viewport performance in complex scenes
  • Startup time: Studio now launches 15% faster on average

New Features

While this is primarily a maintenance release, we've added a few small quality-of-life improvements:

  • Added a template selector when creating new scripts
  • New keyboard shortcut (Ctrl+Shift+P) to open the Plugin Manager
  • Added a "Copy Path" option to the Explorer context menu

Known Issues

We're aware of these issues and working on fixes for the next update:

On some Linux systems, Studio may fail to launch if certain graphics libraries are missing. As a workaround, install the latest Mesa drivers.

For a complete list of changes, see the full changelog.
